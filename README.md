1.Overview
The Bank Balance Tracker is a Java console-based application that tracks daily bank transactions and calculates the running balance using the Prefix Sum technique.
It also supports efficient range sum queries, which helps in finding the total balance change between specific days.

2.Features
Takes daily transaction values as input
Calculates running balance for each day
Uses Prefix Sum for optimized calculations
Finds the sum of transactions in a given range
Simple, beginner-friendly Java implementation.

 3.Technologies Used
Java
Arrays
Prefix Sum Technique
Scanner class for input.

4.Output
Running balance for each day
Sum of transactions between a given range of days.
